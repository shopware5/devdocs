_components

The _components directory contains all styled component-elements of shopware.
Components are typical HTML elements that have inherited styling to get you started. Every example shows you the styling that we've included in our Responsive Theme.

Caution!

Some elements that need JavaScript functions to fully work might not work within this demo.
