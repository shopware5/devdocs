Developer Documentation

_modules

The _modules directory contains all modules elements of shopware.
Modules are mostly larger chunks of code that take care about a specific function, like the shopping cart or whole pages. So they come with templates files and mutual .less file.

Remember!

If you want to edit the template files you need to dublicate the path and the name of the file you want to edit in your own theme. The comments in the examples might help you.

Breadcrumb

Defines the styling for the content breadcrumb elements that display the page location

A breadcrumb displays the path you or a product is in. It's a neat feature to quickly get an idea where you are.
Every breadcrumb entry is followed by a breadcrumb separator list-entry with an arrow icon. The main breadcrumb is positioned below the main navigation bar. 

// Here is the template file located
'yourProject/themes/Frontend/Bare/frontend/index/breadcrumb.tpl'

// frontend/index/index.tpl

Cart

Defines the styles for the shopware 5 cart element

Shopware offers two different shopping cart element options that can be selected in the shopware backend.
There are two cart options available:

  • This is our default shopping cart. It expands on the right side of the page.off-canvas cart
  • This cart looks like a dropdown when opened, similar to the Shopware 4 cart. Which cart you want to use can be configurated in the backend.collapse cart

Global

Defines the global styles of shopware

It contains the global modifier classes that help working with the shopware elements and the typographic styles.

// Global modifier classes

// Appearance
.is--rounded // changes the border-radius to 3px;

// Display
.is--block // sets element to display: block;
.is--inline // sets element to display: inline;
.is--inline-block // sets element to display: inline-block;
.is--hidden // sets element to display: none;
.is--invisible // sets element to visibility: none;

// Text alignment
.is--align-left // sets element to text-align: left;
.is--align-right // sets element to text-align: right;
.is--align-center // sets element to text-align: center;

// Text style
.is--underline // sets element to text-decoration: underline;
.is--line-through // sets element to text-decoration: line-through;
.is--uppercase // sets element to text-transform: uppercase;
.is--strong // sets element to font-weight: 600;
.is--bold // sets element to font-weight: bold;
.is--italic // sets element to font-style: italic;
.is--nowrap // sets element to white-space: nowrap;

// Color
.is--dark // changes color to @text-color-dark;
.is--light // changes color to darken(@gray-dark, 20%);
.is--soft // changes color to @gray-dark;

// Width & position
.is--fluid // sets element to width: 100%;
.left // sets element to float: left;
.right // sets element to float: right;
