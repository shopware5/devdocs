Community

Shopware is open source and wants to involve you as a developer.

For that reason there are several ways to get in touch with us:

  • Shopware forums
  • @shopware on Twitter
  • Issue Tracker
  • Shopware Sourcecode on Github
  • Shopware Labs on Github
  • Crowdin (localization and translation platform)
  • IRC Channel #shopware on Freenode
