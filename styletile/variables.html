Developer Documentation

_variables

The _variables directory contains the imports to all variables shopware uses.
Most variables are controllable by using the backend theme manager.
More precise designations are commented inside the documents.

Remember!

With these variables you can quickly edit the feeling of your whole site. Bear in mind that these can make huge differences if used continuously.

view file

Structure

The variables define the widths for the different viewports

@phoneLandscapeViewportWidth    // Viewport Phone (landscape)    (Default: 480px)
@tabletViewportWidth            // Viewport Tablet               (Default: 768px)
@tabletLandscapeViewportWidth   // Viewport Tablet (landscape)   (Default: 1024px)
@desktopViewportWidth           // Viewport Desktop              (Default: 1260px)

Typography

The variables define the sizes and weights for typographic elements

The values can be changed with the backend theme manager. Find the 'Configurate theme' button and select the tab 'Color configuration'. There are color values as well as typograhpy values defined.

// Base configuration
@font-size-base         // customise in Theme Manager (Default: 14)
@font-base-weight       // customise in Theme Manager (Default: 500)
@font-light-weight      // customise in Theme Manager (Default: 300)
@font-base-stack        // customise in Theme Manager (Default: "Open Sans", "Helvetica Neue", Helvetica, Arial, "Lucida Grande", sans-serif)
@font-headline-stack    // customise in Theme Manager (Default: @font-base-stack)

// Headling font sizes
@font-size-h1           // customise in Theme Manager (Default: 26)
@font-size-h2           // customise in Theme Manager (Default: 21)
@font-size-h3           // customise in Theme Manager (Default: 18)
@font-size-h4           // customise in Theme Manager (Default: 16)
@font-size-h5           // customise in Theme Manager (Default: @font-size-base)
@font-size-h6           // customise in Theme Manager (Default: 12)

// Shopware font directory
@font-directory         // font-directory folder (Default: "../../fonts/")

// Open Sans font directory
@OpenSansPath           // "Open Sans" font-directory folder (Default: "./../fonts/open-sans-fontface")
